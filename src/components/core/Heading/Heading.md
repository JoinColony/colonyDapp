## Headings

### Normal headings

```js
<Heading appearance={{ size: 'huge' }}>Huge heading</Heading>
<Heading appearance={{ size: 'large' }}>Large heading</Heading>
<Heading appearance={{ size: 'medium' }}>Medium heading</Heading>
<Heading appearance={{ size: 'normal' }}>Normal heading</Heading>
<Heading appearance={{ size: 'small' }}>Small heading</Heading>
<Heading appearance={{ size: 'tiny' }}>Tiny heading</Heading>
```

### Invert headings

```js
<div style={{ backgroundColor: 'black' }}>
  <Heading appearance={{ theme: 'invert', size: 'huge' }}>Huge heading</Heading>
  <Heading appearance={{ theme: 'invert', size: 'large' }}>Large heading</Heading>
  <Heading appearance={{ theme: 'invert', size: 'medium' }}>Medium heading</Heading>
  <Heading appearance={{ theme: 'invert', size: 'normal' }}>Normal heading</Heading>
  <Heading appearance={{ theme: 'invert', size: 'small' }}>Small heading</Heading>
  <Heading appearance={{ theme: 'invert', size: 'tiny' }}>Tiny heading</Heading>
</div>
```

### Other heading themes

```js
<Heading appearance={{ theme: 'primary', size: 'huge' }}>Primary heading</Heading>
<Heading appearance={{ theme: 'dark', size: 'huge' }}>Dark heading</Heading>
<Heading appearance={{ theme: 'uppercase', size: 'huge' }}>Uppercase heading</Heading>
```

### Weights

```js
<Heading appearance={{ theme: 'dark', weight: 'bold', size: 'large' }}>Bold heading</Heading>
<Heading appearance={{ theme: 'dark', weight: 'medium', size: 'large' }}>Medium heading</Heading>
<Heading appearance={{ theme: 'dark', weight: 'thin', size: 'large' }}>Thin heading</Heading>
```
